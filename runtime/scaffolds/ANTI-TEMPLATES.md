# Anti Templates

Track the decisions that must stay out of shared scaffolds.

Add one line each time you explicitly choose not to pre-template something.

- Default lint or format rules: language and framework specific.
- Default commit message policy: team preference, not protocol.
- Pre-filled gotchas: only record lessons earned by a real failure.
- Default source tree layout: framework specific.
- Concrete subagent task examples: content layer, not scaffold layer.
